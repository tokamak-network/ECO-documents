# 토카막 재단 서베이 응답

---

## 1. TON 토큰 가격에 긍정적 영향을 미치는 경로

### (b) 비경제적 관점: 인프라 + 기술 경쟁력 + 사용자 확보를 통한 생태계 신뢰도 강화

기존에 제출한 세 가지 기여(Delegate Staking, Zodiac, Eth·RPG)는 각각 다른 계층에서 토카막 생태계의 신뢰도를 높이며, 이것이 TON 가격과 시총 상승에 기여하는 경로입니다.


[Delegate Staking → 네트워크 안정성 → 홀더 신뢰]

Delegate Staking은 V3 핵심 인프라로, 일반 사용자가 TON을 시퀀서에 위임하여 시뇨리지 보상을 받을 수 있게 합니다. 이것이 TON 가격에 기여하는 경로:

- 더 많은 사용자가 스테이킹에 참여할수록 네트워크 보안과 탈중앙화가 강화됩니다
- 스테이킹 참여는 홀더의 장기 보유 의지를 나타내는 시그널로, 네트워크에 대한 시장 신뢰를 강화합니다
- "일반 사용자도 보상을 받을 수 있다"는 것은 TON 보유의 실질적 이유를 제공하여 홀더 이탈을 방지합니다
- 이 인프라가 없으면 V3에서 Bridged TON으로 시뇨리지가 전환될 때 일반 홀더가 보상 경로를 잃게 되어, 토큰 보유 동기가 약해집니다


[Zodiac → 기술 경쟁력 → 시장 차별화]

Zodiac은 ZK 기반 분쟁 프로토콜 PoC로, 롤업의 7일 분쟁 기간을 1-2시간으로 단축합니다. 이것이 TON 가격에 기여하는 경로:

- L2 finality 속도는 경쟁 L2(Optimism, Arbitrum 등)와의 핵심 차별화 요소입니다
- "7일 → 1-2시간"이라는 수치는 투자자와 개발자에게 명확한 기술적 우위를 전달합니다
- 기술적 차별화가 있는 L2는 개발자와 프로젝트를 유치하기 쉬우며, 이는 생태계 성장 → 토큰 가치 상승으로 연결됩니다
- 이것이 없으면 토카막 L2는 7일 분쟁이라는 동일한 병목을 가진 수많은 L2 중 하나로 남게 됩니다


[Eth·RPG → 사용자 유입 → 생태계 활성화]

Eth·RPG는 이더리움 지갑을 RPG 캐릭터로 변환하는 바이럴 서비스로, 토카막 L2에서 NFT 민팅과 온체인 배틀을 통해 실제 사용자를 유입시키는 것이 목표입니다. 이것이 TON 가격에 기여하는 경로:

- 지갑 주소만 입력하면 카드가 생성되고 SNS로 공유하는 바이럴 루프는, 광고비 없이 작동하는 오가닉 사용자 유입 채널입니다
- 토카막 L2 런칭 시 "이미 만들어진 사용자 기반"을 가진 dApp이 있다는 것은, L2의 초기 활성화(cold start)를 해결하는 핵심 자산입니다
- 실제 사용자가 있는 생태계는 투자자/시장에 "이 네트워크는 쓰이고 있다"는 신호를 보내며, 이는 직접적으로 시장 신뢰도에 영향을 미칩니다

현재 프로덕트 실측 데이터 (6일 만에 프로덕션 배포 완료):
- 소스 파일 141개, 코드 약 20,000줄, API 11개, 컴포넌트 24개, 테스트 38개
- 원시 트랜잭션을 34개 프로토콜 기준으로 분류하고, 7개 스탯 통계 모델을 적용하고, AI로 캐릭터 스토리를 생성하는 end-to-end 파이프라인 완성


[세 프로젝트의 시너지 → TON 가격 경로]

이 세 가지는 독립적이 아니라 연결됩니다:

Delegate Staking(홀더 유지) + Zodiac(기술 경쟁력) + Eth·RPG(사용자 유입) → 토카막 생태계의 인프라/기술/사용자 세 축이 모두 강화 → 시장 신뢰도 상승 → 투자자/개발자/사용자 유입 → TON 수요 및 시총 상승

이 중 Delegate Staking은 V3 인프라 개발이 진행 중이고, Zodiac은 PoC가 완료되었으며, Eth·RPG는 프로덕션 배포가 완료되어 마케팅 런칭 단계입니다. 향후 30일은 이 세 프로젝트의 진행과 검증에 집중합니다.

## 2. 향후 30일 구체적 확장 및 운영 계획

향후 30일의 최우선 목표는 Eth·RPG의 시장 검증입니다. 프로덕션 배포는 완료되었으나, "만든 것이 사용자에게 가치가 있는가"는 아직 확인되지 않았습니다. 이것을 데이터로 검증하는 것이 이번 30일의 핵심입니다. Delegate Staking과 Zodiac은 병행 진행하되, 주력은 Eth·RPG입니다.


[Week 1 (Day 1-7): 런칭 준비 + 마일스톤 설정]

- Eth·RPG 마케팅 런칭 준비: 랜딩 최적화, OG 이미지 점검, 공유 플로우 테스트 → 런칭 체크리스트 완료
- Delegate Staking / Zodiac 현재 진행 상태 정리 및 다음 마일스톤 설정


[Week 2 (Day 8-14): Eth·RPG 마케팅 런칭 + 사용자 데이터 수집]

- Eth·RPG 커뮤니티 시딩 시작: Crypto Twitter, Farcaster, 텔레그램 20+곳 → 시딩 리포트
- 유명 지갑 분석 콘텐츠 제작 (vitalik.eth 등) → 바이럴 시드용 SNS 콘텐츠 10+개
- 사용자 행동 추적 설정 (Vercel Analytics 또는 이벤트 로깅) → 데이터 수집 파이프라인
- Delegate Staking / Zodiac 개발 진행


[Week 3 (Day 15-21): 데이터 기반 개선 + 토카막 연동]

- Eth·RPG Week 2 사용자 데이터 분석, 피드백 기반 개선 1-2회 배포
- Eth·RPG에 토카막 브랜딩/연동 추가 (Powered by Tokamak, 토카막 L2 배포 계획 안내 등)
- Delegate Staking / Zodiac 개발 진행 및 중간 리뷰


[Week 4 (Day 22-30): 측정 & 성과 정리]

- 전체 기간 사용자 데이터 종합 분석 → 데이터 대시보드
- 세 프로젝트 통합 진행 상황 리포트
- 30일 종합 성과 리포트 → 최종 리포트


[30일 KPI]

Eth·RPG (검증 단계):
- 커뮤니티 시딩: 20+곳 게시
- SNS 콘텐츠: 30+개
- 총 노출: 2,000-4,000회 (타겟 커뮤니티 평균 규모 1,000-2,000명, 게시물 도달률 10% 가정)
- 사이트 방문자: 200-400명 (노출 대비 10% 클릭율 가정)
- 카드 생성 수: 30-80건 (방문자 대비 15-20% 전환율 가정, 지갑 주소 입력의 심리적 장벽 고려)

Delegate Staking / Zodiac (병행):
- 각 프로젝트 다음 마일스톤 진행 및 상황 공유

숫자가 작아 보일 수 있으나, 이것은 의도적으로 보수적인 추정입니다. 작지만 현실적인 목표를 달성하고 거기서 배우는 것이, 큰 숫자를 제시하고 미달하는 것보다 낫습니다. 각 수치는 실제 데이터가 쌓이면 Week 3에서 상향 또는 하향 조정합니다.


[실패 대응 플랜]

- Week 2 시딩 반응이 낮을 경우 → 타겟 커뮤니티를 재선정하고, 콘텐츠 형식을 텍스트에서 시각 중심(카드 이미지 비교, 인포그래픽)으로 전환. 영상/GIF는 Week 1에서 템플릿을 미리 준비해둠
- 방문자 대비 카드 생성 전환율이 15% 미만일 경우 → 이탈 지점 분석, 지갑 주소 예시 제공으로 진입 장벽 완화, 원클릭 샘플 카드 기능 추가 검토
- 전체적으로 목표 미달 시 → Week 4 리포트에 원인 분석 포함, "무엇이 작동하지 않았는가"를 데이터로 정리하고, 수정된 다음 30일 계획을 제시

## 3. 상업적 검증 (외부자 관점)

외부자 입장에서 향후 30일간의 산출물을 토카막 재단에 판매한다고 가정합니다. 핵심 산출물은 Eth·RPG 서비스 운영 및 마케팅 런칭입니다.


[30일간 산출물 목록]

1. Eth·RPG 라이브 서비스 운영 + 마케팅 런칭 실행 — 프로덕션 배포 완료된 20,000 LOC 규모 웹 서비스 + 시딩 캠페인
2. 30일 성과 분석 리포트 — 사용자 데이터 대시보드 + 검증 결과 문서


[판매 모델]

일회성 프로젝트 납품 (One-time Product Sale). 산출물은 납품 가능한 완성품이므로, 프로젝트 단위 납품이 적합합니다.


[가격 설정 및 근거]

시장 가격 기준:

커스텀 dApp 개발 + 런칭 실행 1건: $25,000 - $45,000
→ Web3 전문 에이전시 평균 단가. Toptal 시니어 풀스택($100-$150/hr) 기준 $48,000-$72,000. 프리랜서 마켓(Upwork) 기준 $20,000-$40,000. 양쪽 범위의 교집합 구간 적용.

데이터 분석 리포트: $1,000 - $2,000
→ 데이터 분석 컨설팅 단가 기준.

합계: $26,000 - $47,000


[제안 가격]

$25,000 (일괄 납품). 내부 기여자 할인과 재단 예산 현실을 고려한 가격입니다.


[현재 $5,000/월과의 관계]

30일 산출물의 시장가는 $26,000-$47,000입니다. 현재 월 $5,000을 받고 있으므로, 재단은 시장가의 약 1/5~1/9 비용으로 동일한 산출물을 확보하고 있는 셈입니다. 이것은 "사람을 싸게 쓰고 있다"는 이야기가 아닙니다. "동일한 결과물을 외부에서 조달하면 5배 이상 비싸다"는 객관적 비교입니다.


[설득 논리]

외부 에이전시에 동일한 산출물을 발주하면 최소 $25,000 이상이며, 기간은 2-3개월입니다. 저는 30일 안에 동일한 퀄리티를 납품할 수 있습니다. 근거는 추정이 아니라 실적입니다 — Eth·RPG 하나만으로도 6일간 93커밋, 141파일, 20,000줄의 프로덕션 레벨 서비스를 배포했습니다.


[재단이 구매하지 않을 경우]

계속할 수 있습니다. 그러나 재단과 함께할 때 시너지가 극대화됩니다.

계속할 수 있는 이유:
- Eth·RPG는 독립적으로 운영 가능합니다. 다른 생태계 그랜트(Arbitrum, Optimism RPGF, Base Ecosystem Fund)에 같은 산출물로 신청할 수 있습니다.
- AI 활용 개발 역량은 다른 Web3 생태계에서도 동일하게 적용 가능합니다.

그러나 재단과 함께하는 것이 더 나은 이유:
- Delegate Staking과 Zodiac은 토카막 전용 인프라입니다. 다른 생태계에서는 의미가 없으며, 토카막 안에서만 가치를 발휘합니다.
- 세 프로젝트의 시너지(인프라 + 기술 + 사용자)는 토카막 생태계 안에서만 완성됩니다. 하나씩 떼어서는 전체 그림이 만들어지지 않습니다.
- 팀 내 AI 역량 전파는 재단 내부에서만 가능합니다. 이것이 장기적으로 가장 큰 가치이며, 외부에서는 실행할 수 없는 기여입니다.

중단하는 경우: 재단의 전략적 방향과 제 역량이 완전히 불일치하여 시너지가 불가능할 때. 이 경우 서로의 시간을 절약하는 것이 합리적입니다.
