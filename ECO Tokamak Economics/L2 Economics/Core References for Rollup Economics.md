- **Rollups as Layer 2 Solutions:**
  - Bundle off-chain transactions and post compressed data on Ethereum’s L1.
  - Balance scalability with security by leveraging L1’s infrastructure.
- **Key Players:**
  - **Users:** Pay transaction fees to use the network.
  - **Sequencers (Operators):** Aggregate and process transactions, then post data on L1.
  - **L1 (Ethereum):** Provides security and data availability.
- **Revenue Sources:**
  - Transaction fees.
  - MEV (Maximal Extractable Value) extraction.
  - Token-related mechanisms.
- **Cost Components:**
  - Operational expenses (node management, transaction sequencing).
  - Data publishing costs (reduced significantly by upgrades like EIP‑4844).
  - Proof generation costs (varies between ZK rollups and Optimistic rollups).
- **Impact of EIP‑4844:**
  - Changes data publication from calldata to blobs, reducing costs by over 90%.
  - Reshapes the rollup economic model and enables future innovations (e.g., decentralized sequencing).
