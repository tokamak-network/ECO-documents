현재 앱에서 팀원들이 실행해보려면 로컬에서 환경을 세팅하거나 Sepolia에서 실행해야 합니다. 로컬 환경은 직접 설치해야 하는 불편함이 있고, Sepolia에서는 timetravel이 불가능합니다.

이 문제를 해결하기 위해 Anvil 기반 샌드박스 환경을 제공할 예정입니다. "샌드박스 환경에서 실행해보기" 버튼(해당 버튼은 지갑이 연결되어 있어야 가능할 것 같습니다.)을 클릭하면 fly.io에 샌드박스 환경이 자동으로 세팅됩니다. 이후 연결된 지갑에 테스트용 faucet을 제공하고, 샌드박스 환경의 Anvil과 자동으로 연결됩니다. 샌드박스 환경 구축 후 Anvil 네트워크에 연결되면 누구나 쉽게 데모를 진행할 수 있습니다.

우선은 Sepolia에서 테스트 부탁드릴게요. 샌드박스 환경에서 테스트 해볼 수 있도록 환경은 구축했는데 Vercel 환경변수 설정 권한이 저에게 없어서 내일 오전 중에 요청드리고 다시 말씀드릴게요!

클라우드 anvil 배포해서 사용할 수 있도록 환경 구축은 완료했는데, 클라우드 내의 anvil 로컬 체인과 지갑 연결이 잘되지 않는 것 같아요. 

샌드박스 환경에서 테스트할 수 있도록 환경은 구축했는데, 샌드박스 내의 로컬 블록체인(anvil)과 지갑 연결이 잘 되지 않는 문제가 있네요ㅜ. 지갑 문제인지 클라우드(fly.io) 상의 문제인지 내일 오전까지 해결해보고 다시 말씀드릴게요.

우선 사용해보고 싶으신 분은 Sepolia에서 사용 부탁드릴게요. (Vercel 환경변수 설정 권한이 저에게 없어서 내일 오전중으로 요청드리고 이 문제는 해결해보도록 할게요.)
