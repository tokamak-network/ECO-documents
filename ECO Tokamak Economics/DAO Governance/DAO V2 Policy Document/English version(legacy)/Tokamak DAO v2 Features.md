# What’s new in Tokamak DAO v2?

- **Streamlined Governance**
  - Simplified workflow from proposal to execution.
- **Introduction of the Security Council**
  - A 3-member DAO committee is now named the Security Council.
- **Extensive Example Agendas**
  - The community can reference example DAO agendas for better engagement and understanding.
- **Migration of DAO Frontend to Tally**
  - Transitioning the DAO frontend to Tally for enhanced governance functionality.
- **Voting for Staked TON Holders**
  - Allowing simple TON stakers (Mint and Delegate) to participate in voting on proposals and agendas.
