시나리오 A: 전체 거버넌스 플로우 (15-20분)

1단계: 환경 설정 (Faucet)

- /faucet 페이지에서 vTON, TON 토큰 수령
- Sepolia ETH도 확보 (가스비용)

2단계: 델리게이트 등록

- /delegates 페이지에서 "델리게이트 되기" 클릭
- 프로필, 투표 철학, 관심사 입력
- 트랜잭션 확인

3단계: 위임 (Delegation)

- 다른 지갑(또는 시연용 계정)에서 vTON을 방금 등록한 델리게이트에게 위임
- 위임 제한(20% 캡), 7일 최소 기간 설명

4단계: 프로포절 생성

- /proposals/create에서 새 프로포절 작성
- 100 TON 소각됨을 보여줌

5단계: 투표

- 활성화된 프로포절에서 For/Against/Abstain 투표
- 투표 결과 실시간 확인

6단계: Security Council (선택)

- 긴급 상황 대응 기능 시연
- 2/3 다수결 메커니즘 설명
