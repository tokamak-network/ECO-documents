# **DMV-01 | `minDepositGasLimit` Is Not Used**

| **카테고리** | **심각도** | **위치** | **상태** |
| --- | --- | --- | --- |
| **Logical-issue** | **사소한** | **DepositManagerV1_1.sol#L103-L106: L103-L106DepositManagerV1_1.sol#L157-L158: L157-L158DepositManagerV1_1.sol#L180-L180: L180-L180DepositManagerV1_1.sol#L195-L195: L195-L195DepositManagerV1_1.sol#L204-L204: L204-L204** | **Pending** |

### **Description**

The `DepositManagerV1_1` contract contains a `minDepositGasLimit` setter function, but the value of `minDepositGasLimit` is never utilized outside of the setter function. In the `withdrawAndDepositL2()` function, a different local variable `_minDepositGasLimit` is declared and utilized instead.

### **Recommendation**

Consider removing unused setter function and / or update the logic in the `withdrawAndDepositL2()` function to use the `minDepositGasLimit` variable instead.
